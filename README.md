# MemoryBooster
基于beego搭建的类百词斩安卓软件后台

## 快速入门

### 简介
MemoryBooster是一个基于beego开发的服务器后台，包含登录验证和前后端数据交互

### model
models包中存放了模型

### controller
控制器存放在controller包中

### mtool
mtool包中存放了一些小工具
目前包含了数据库连接工具、邮件发送工具、uuid生成工具、json操作工具和token操作工具

### router
路由存放在router包中

### static
static包中存放了静态文件
对于本项目来说，static包用于存放用户的dict文件
